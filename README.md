### 실행 방법
   1. style button으로 자신이 원하는 모델 스타일을 선택한다.
   2. 자신의 이름을 입력한다.
   3. click-photo button으로 사진을 찍는다.
   4. 총 3개의 사진을 찍는다. 
   5. Result image button을 눌러 찍은 3개의 이미지를 확인한다.
   6. 3개의 이미지중 제일 마음에 드는 사진을 선택한다.
